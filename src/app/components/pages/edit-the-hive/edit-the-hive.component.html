Edit the Hive

- To add users if needed.

a) Click on three vertical dots to the right of the hive name.

b) Click on “Edit Hive”

c) Click on “+ Add User”

d) Enter the email address of the person you are adding and click Search.

e) Select the user you are adding. The default user level is “Member” which can only add documents to the hive.

f) Change to Read Only, Manager, etc. if necessary by clicking the down-arrow next to manager and selecting from the drop-down box.

g) Click on “Update Hive” to save your changes.
