Definitions

Some commonly used terms throughout BeeHive

There are several components to BeeHive which are defined below:

Templates -

A reusable form which prompts employees with action steps to follow and questions to answer. This template can be reused for multiple items or purposes.

Example – “End of Season Checklist.”

Items -

These are the various pieces of equipment or areas which will be examined according to the steps outlined in the Template above.

Multiple Items can reference the same Template.

Example – Lawn mower, motor boat, and Jet Ski can be three separate Items all using the same Template “End of Season Checklist.”

Procedures -

A procedure walks the user through a series of actions and/or questions examining the necessary items and in a particular order if appropriate.

Example – “Prepare for Winter” could be a procedure which includes three Items – Lawn Mower, Motor Boat, and Jet Ski.

Items -

A catalog enables the user to access one specific Item to examine versus accessing the Item in a specified order as part of a larger Procedure.
