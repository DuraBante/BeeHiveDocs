What is BeeHive?

BeeHive is a configurable data application that can fit into any data collection and record keeping process. In the application you create a Hive which is a repository of templates, items, and procedures customized to your organization and objective. Each item can be assigned multiple templates to define what information you want to collect. The instructions below will outline the process involved.
